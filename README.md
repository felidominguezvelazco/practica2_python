# Práctica 2 - Python

## Requisitos
- Python 3.x
- Jupyter Notebook

## Instalación

1. Clonar el repositorio:
git clone <URL_DEL_REPO>

2. Crear un entorno virtual:
python -m venv venv

3. Activar el entorno virtual:

En Windows:
venv\Scripts\activate

En Linux/Mac:
source venv/bin/activate

4. Instalar dependencias:
pip install -r requirements.txt

## Ejecución

1. Abrir Jupyter Notebook:
jupyter notebook

2. Ir a la carpeta `notebooks`

3. Ejecutar los notebooks correspondientes a cada ejercicio

## Estructura del proyecto

- `src/`: contiene las funciones y lógica de los ejercicios
- `notebooks/`: contiene los notebooks para ejecutar y probar los ejercicios
